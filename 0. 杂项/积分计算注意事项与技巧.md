# 注意事项

-  $\int \frac{1}{x} \mathrm{d}x = \ln |x| + C$
	使用这个公式时要注意绝对值
- [[数一\1 高数\4 一元函数积分学及其应用\1. 概念\不定积分|不定积分]]计算最后要 $+C$
- $\int x^{k} \mathrm{d}x = \frac{1}{k + 1} x^{k + 1} + C, k \neq -1$
	- 这个公式的条件是 $k \neq -1$
