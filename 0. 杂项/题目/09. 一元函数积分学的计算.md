### 27.
- 题型：含定积分的函数方程，求 $f(x)$ 表达式
	- [灵光乍现](0.%20杂项/灵光乍现.md#含积分%20极限的函数方程)
- **设常数**：设 $\int_0^1 f(x)dx = A$（注意：积分部分是常数）
- **代回原方程**：$f(x) = \frac{1}{1+x^2} + Ax^3$
- **两边积分**：$\int_0^1 f(x)dx = \int_0^1 \frac{1}{1+x^2}dx + \int_0^1 Ax^3 dx$
- **计算**：$A = \arctan x\big|_0^1 + \frac{A}{4}x^4\big|_0^1 = \frac{\pi}{4} + \frac{A}{4}$
- **解方程**：$A - \frac{A}{4} = \frac{3A}{4} = \frac{\pi}{4}$，故 $A = \frac{\pi}{3}$
- **代回**：$f(x) = \frac{1}{1+x^2} + \frac{\pi}{3}x^3$
- **核心技巧**：设 $\int_a^b f(x)dx = A$ 为常数，两边积分构造方程求 $A$
