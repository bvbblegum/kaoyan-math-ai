# 学习进度

> [!tip] 使用方法
> 每学完一个知识点，在该笔记的 frontmatter 中添加 `掌握度: ★★★☆☆`（1-5星）
> 本页面会自动统计你的学习进度。

## 高等数学

```dataview
TABLE WITHOUT ID
  file.link as "章节",
  length(rows) as "笔记数",
  length(filter(rows, (r) => r.掌握度)) as "已评估",
  choice(length(filter(rows, (r) => r.掌握度 >= 4)) > 0, "✅", "📝") as "状态"
FROM "数一/1 高数"
WHERE 属性 = "正文"
GROUP BY split(file.folder, "/")[3] as chapter
SORT chapter
```

## 错题记录

```dataview
TABLE file.link as "错题", 科目, 章节, 错误原因
FROM "数一"
WHERE 属性 = "错题"
SORT file.ctime DESC
LIMIT 20
```
