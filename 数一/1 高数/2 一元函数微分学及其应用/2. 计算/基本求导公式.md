- $(x^\alpha)’ = \alpha x^{\alpha - 1}$
- $(\alpha^x)’ = \alpha^x \ln \alpha~(\alpha > 0, \alpha \ne 1)$
- $(a^x)^{(n)}=a^x(\ln a)^n$
- $(e^x)’ = e^x$
- $(\log_\alpha x)’ = \frac{1}{x\ln \alpha}~(\alpha > 0, \alpha \ne 1)$
- $\left( \frac{1}{x} \right)'=-\frac{1}{x^2}(x\neq_{0})$
- $(\sqrt{ x })'=\frac{1}{2\sqrt{ x }}$

- $(\ln |x|)’ = \frac{1}{x}~(x \ne 0)$ ^14tsh7

> [!note] $\ln|x|$ 求导，视绝对值而不见！
> $(\ln x)'=\frac{1}{x}$
> $(\ln (-x))'=\frac{1}{-x}\times(-1)=\frac{1}{x}$
> 求导后结果相同

- $(\cot x)’ = -\csc^2 x$
