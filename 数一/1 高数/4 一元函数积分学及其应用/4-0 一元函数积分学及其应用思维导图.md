# 一元函数积分学
## 不定积分
- ### 不定积分的概念与性质
	- #### 原函数与不定积分的定义
		- **原函数定义**
			- 若 $F'(x) = f(x)$，则称 $F(x)$ 是 $f(x)$ 的一个原函数。
		- **不定积分定义**
			- $\int f(x)dx = F(x) + C$
		- **几何意义**
			- 表示一族积分曲线。
		- **不定积分与导数的关系**
			- $\frac{d}{dx}[\int f(x)dx] = f(x)$
			- $\int F'(x)dx = F(x) + C$
		- [[原函数(不定积分)存在定理]]
			- `连续`函数 $f(x)$ 必有原函数 $F(x)$
			- 含有`第一类间断点`和`无穷间断点`的函数 $f(x)$ 在包含该间断点的区间内必没有原函数 $F(x)$
			- `振荡间断点`可能有
		- ==原函数一定连续==

## 定积分
- ### 定积分的概念与性质
	- #### 定积分的定义
		- 定义式: $\int_a^b f(x)dx = \lim_{\lambda \to 0} \sum_{i=1}^n f(\xi_i)\Delta x_i$
		- 几何意义: 表示`曲边梯形`的面积。
		- [[定积分存在定理]]
			- 充分条件
				- 在 $[a,b]$ 上`连续`。
				- 在 $[a,b]$ 上`有界`且只有`有限个间断点`
				- 在 $[a,b]$ 上仅有**有限个**`第一类间断点`
				- 在 $[a,b]$ 上`单调`
			- 必要条件
				- 定积分存在一定`有界`
			- 不定积分与定积分的存在性无必然联系 ![[4-0 一元函数积分学及其应用思维导图-1.png]]
	- #### 定积分的性质
		- **保号性**: 若 $f(x) \ge 0$, 则 $\int_a^b f(x)dx \ge 0$
		- 比较定理: 若 $f(x) \le g(x)$, 则 $\int_a^b f(x)dx \le \int_a^b g(x)dx$
		- 估值定理: $m(b-a) \le \int_a^b f(x)dx \le M(b-a)$
		- [[积分中值定理]]: $\exists \xi \in [a,b]$, s.t. $\int_a^b f(x)dx = f(\xi)(b-a)$
- ### 变限积分及其导数
	- 定义：设 $f(x)$ 在 $[a,b]$ 上`连续`，则 $\int_a^x f(t)\,dt$ 在 $[a,b]$ 上`可导`，且：  $\left( \int_a^x f(t)\,dt \right)' = f(x)$  
	- 变限积分性质：
		- 连续性：若 $f(x)$ 在 $[a,b]$ 上可积，则  $\int_a^x f(t)\,\text{d}t$  在 $[a,b]$ 上`连续`。  
		- 可导性：若 $f(x)$ 在 $[a,b]$ 上除点 $x = x_0 \in (a,b)$ 外均连续，则在点 $x = x_0$ 处： 
			- 连续：可导，且 $\boldsymbol{F'(x_0) = f(x_0)}$
			- `可去`：`可导`，且 $\boldsymbol{F'(x_0) = \lim\limits_{x \to x_0} f(x)}$
			- `跳跃`：**连续但不可导**，且 $\boldsymbol{F'_-(x_0) = f(x_0^-)}$，$\boldsymbol{F'_+(x_0) = f(x_0^+)}$
		- 奇偶性:设 $f(x)$ 连续：  
			- 若 $f(x)$ 为`奇`函数，则  $\int_a^x f(t)\,\text{d}t$  为`偶`函数。  
			- 若 $f(x)$ 为偶函数，则  $\int_0^x f(t)\,\text{d}t$  为奇函数。  
		- 有条件的周期性:设 $f(x)$ `连续`且以 $T$ 为周期
			- 则 $F(x) = \int_0^x f(t)dt$ 是以 $T$ 为周期的周期函数 $\iff \int_0^T f(x)dx = 0$。
			- 恒有： $\displaystyle\int_{x}^{x+T}f(t)dt=\int_{0}^Tf(t)dt$
		    - 恒有： $\displaystyle\int_{0}^{nT}f(x)dx=n\int_{0}^Tf(x)dx$

## 反常积分
#### 无穷区间的反常积分
- 定义：
	- $\displaystyle\int_{a}^{+\infty}f(x)dx=\lim_{t\to+\infty}\int_{a}^{t}f(x)dx$ 若这个极限存在，则称这个反常积分是收敛的；反之则为发散的；
	- $\displaystyle\int_{-\infty}^{+\infty}f(x)dx=\int_{-\infty}^{0}f(x)dx+\int_{0}^{+\infty}f(x)dx$ 若这两个都收敛，则收敛；否则发散；
- 比较判别法
	- [[比较判别法#无穷区间]]
#### 无界函数的反常积分 (瑕积分)
- 定义: 
	- 如果函数 $f(x)$ 在点 a 的任意邻域内都无界，则点 a 为函数 $f(x)$ 的瑕点：$\displaystyle\int_a^bf(x)dx=\lim_{t\to a^+}\int_t^bf(x)dx$
	- 设点 b 为函数 $f(x)$ 的瑕点 $\displaystyle\int_{a}^{b}f(x)dx=\lim_{t\to b^{-}}\int_{a}^{t}f(x)dx.$
	- 设点 c 为函数 $f(x)$ 的瑕点 $\displaystyle(a<c<b)\int_{a}^{b}f(x)dx=\int_{a}^{c}f(x)dx+\int_{c}^{b}f(x)dx$
- 比较判别法

## 定积分的应用
- ### 几何应用
	- #### 平面图形的面积
		- 若平面域 $D$ 由曲线 $y=f(x), y=g(x) \ (f(x) \ge g(x))$，及直线 $x=a, x=b \ (a<b)$ 所围成，则 $\displaystyle S = \int_a^b [f(x) - g(x)]dx$
		 - 若平面域 $D$ 由曲线 $r=r(\theta), \theta=\alpha, \theta=\beta \ (\alpha < \beta)$ 所围成，则其面积为 $\displaystyle S =  \int_\alpha^\beta d\theta \int_0^{r(\theta)} r dr = \frac{1}{2}\int_\alpha^\beta r^2(\theta)d\theta$
		- **参数方程**:
			- $S = \int_\alpha^\beta y(t)x'(t)dt$
	- #### 旋转体体积
		- $\displaystyle r(x,y) = \frac{|ax+by+c|}{\sqrt{a^2+b^2}},V = 2\pi \iint_D r(x,y)d\sigma$
		- 区域 D 绕 x 轴旋转 ($r(x,y)=y$) 一周所得旋转体的体积为 $V_x = 2\pi \iint_D y d\sigma = 2\pi \int_a^b dx \int_0^{f(x)} y dy = \pi \int_a^b f^2(x) dx$
		- 区域 D 绕 y 轴旋转 ($r(x,y)=x$) 一周所得旋转体的体积为$V_y = 2\pi \iint_D x d\sigma = 2\pi \int_a^b dx \int_0^{f(x)} x dy = 2\pi \int_a^b xf(x) dx$
	- #### 平面曲线的弧长
		- **直角坐标**: $s = \int_a^b \sqrt{1 + [y'(x)]^2}dx$
		- **极坐标**: $s = \int_\alpha^\beta \sqrt{[r(\theta)]^2 + [r'(\theta)]^2}d\theta$
		- **参数方程**: $s = \int_\alpha^\beta \sqrt{[x'(t)]^2 + [y'(t)]^2}dt$
	- #### 旋转体侧面积
		- $S = 2\pi \int_a^b f(x)\sqrt{1+f'^2(x)}dx$
		- $S = 2\pi \int_a^b |y(x)|\sqrt{1+(y'(x))^2}dx$
		- $S = 2\pi \int_\alpha^\beta |y(t)|\sqrt{(x'(t))^2+(y'(t))^2}dt$
		- $S = 2\pi \int_\alpha^\beta |r(\theta)\sin\theta|\sqrt{r^2(\theta)+[r'(\theta)]^2}d\theta$
 - #### 考研数学常考的几类曲线
     - **1. 摆线 (Cycloid)**
         - **参数方程**: $\begin{cases} x = a(t - \sin t) \\ y = a(1 - \cos t) \end{cases}$
     - **2. 星形线 (Astroid)**
         - **参数方程**: $\begin{cases} x = a\cos^3 t \\ y = a\sin^3 t \end{cases}$
         - **直角坐标**: $x^{2/3} + y^{2/3} = a^{2/3}$
     - **3. 心形线 (Cardioid)**
         - **极坐标方程**: $r = a(1 - \cos\theta)$ (或 $1+\cos\theta, 1\pm\sin\theta$)
     - **4. 双纽线 (Lemniscate)**
         - **极坐标方程**: $r^2 = a^2 \cos(2\theta)$
         - **直角坐标**: $(x^2+y^2)^2 = a^2(x^2-y^2)$
     - **5. 对数螺线 (Logarithmic Spiral)**
         - **极坐标方程**: $r = ae^{b\theta}$
 
- ### 物理应用
	- #### 变力做功
		- $W = \int_a^b F(x)dx$
	- #### 水压力
		- $P = \int_c^d \rho g (h-y) w(y) dy$
	- #### 质心/形心
		- $\bar{x} = \frac{\int_a^b x \cdot dm}{\int_a^b dm}$
		- $\bar{y} = \frac{\int_a^b y \cdot dm}{\int_a^b dm}$
