## 定义
设 $f(x)$ 在 $[a,b]$ 上连续，则 $\int_a^x f(t)\,dt$ 在 $[a,b]$ 上可导，且：  $\left( \int_a^x f(t)\,dt \right)' = f(x)$  

## 性质
### 1. 连续性  
若 $f(x)$ 在 $[a,b]$ 上可积，则  $\int_a^x f(t)\,\text{d}t$  在 $[a,b]$ 上连续。  

### 2. 可导性  
有关 $\boldsymbol{F(x) = \int_a^x f(t)\,\text{d}t}$ 的可导性结论：  
若 $f(x)$ 在 $[a,b]$ 上除点 $x = x_0 \in (a,b)$ 外均连续，则在点 $x = x_0$ 处：  

| $f(x)$ 的性质 | $F(x)$ 的性质                                                                       |
| ---------- | -------------------------------------------------------------------------------- |
| （1）连续      | 可导，且 $\boldsymbol{F'(x_0) = f(x_0)}$                                             |
| （2）可去      | 可导，且 $\boldsymbol{F'(x_0) = \lim\limits_{x \to x_0} f(x)}$                       |
| （3）跳跃      | 连续但不可导，且 $\boldsymbol{F'_-(x_0) = f(x_0^-)}$，$\boldsymbol{F'_+(x_0) = f(x_0^+)}$ |
|            |                                                                                  |
> **联想**：[[定积分存在定理]]与[[原函数(不定积分)存在定理]]，只有 $f(x)$ 在 $[a,b]$ 上连续，$\int_a^x f(t)\,dt$ 才是 $f(x)$ 的原函数
### 3. 奇偶性  
设 $f(x)$ 连续：  
（1）若 $f(x)$ 为奇函数，则  $\int_a^x f(t)\,\text{d}t$  为偶函数。  
（2）若 $f(x)$ 为偶函数，则  $\int_0^x f(t)\,\text{d}t$  为奇函数。  
### 4. 有条件的周期性
- 设 $f(x)$ `连续`且以 $T$ 为周期，则 $F(x) = \int_0^x f(t)dt$ 是以 $T$ 为周期的周期函数 $\iff \int_0^T f(x)dx = 0$。
- $\displaystyle\int_{a}^{a+T}f(x)dx=\int_{0}^Tf(x)dx$
- $\displaystyle\int_{0}^{nT}f(x)dx=n\int_{0}^Tf(x)dx$ ^dk6n0r
