## 定义 
设 A 是 n 阶矩阵, 如果存在 n 阶矩阵 B 使得 $$
AB = BA = E(单位矩阵)
$$成立, 则称 A 是**可逆矩阵**或**非奇异矩阵**, B 是 A 的**逆矩阵**, 记成 $A^{-1} = B$.

若 A 可逆, 则 A 的逆矩阵唯一.
A 可逆 $\Leftrightarrow |A| \neq 0$.
设 A 和 B 是 n 阶矩阵且 AB = E, 则 BA = E.

## n 阶矩阵 A 可逆的充分必要条件

> [!tip] n 阶矩阵 A 可逆的充分必要条件
> 1. 存在 n 阶矩阵 B, 使 $AB = E$ (或 $BA = E$).
> 2. $|A| \neq 0$, 或秩 $r(A) = n$, 或 A 的列 (行) 向量线性无关.
> 3. 齐次方程组 $Ax = 0$ 只有零解.
> 4. $\forall b$, [[数一\2 线代\4 线性方程组\非齐次线性方程组|非齐次线性方程组]] $Ax = b$ 总有唯一解.
> 5. 矩阵 A 的特征值全不为 0. ^nof4lw

## 逆矩阵的运算性质
- 若 $k \neq 0$, A 可逆, 则 $(kA)^{-1} = \frac{1}{k}A^{-1}$.
- 若 A, B 可逆, 则 $(AB)^{-1} = B^{-1}A^{-1}$, 特别地 $(A^2)^{-1} = (A^{-1})^2$.
- 若 $A^T$ 可逆, 则 $(A^T)^{-1} = (A^{-1})^T$; $(A^{-1})^{-1} = A$; $|A^{-1}| = \frac{1}{|A|}$.
- $AB=BA=E$ ^1

> **注意** 即使 A, B 和 A+B 都可逆, 一般地 $(A+B)^{-1} \neq A^{-1} + B^{-1}$.

> [!help]
> $$(A^{-1}+B^{-1})^{-1} = A(A+B)^{-1}B = B(A+B)^{-1}A$$

## 判断是否可逆
1. 行列式不为 0
2. 构造等式，使 $\displaystyle AB=E$ 或 $BA=E$, 则 A 可逆, 且 $A^{-1}=B$
3. [[5-1 特征值与特征向量|特征值与特征向量]]，特征值不含 0
4. 分块，不成比例

## 求逆矩阵的方法
- **方法一** 用公式, 若 $|A| \neq 0$, 则 $\displaystyle A^{-1} = \frac{1}{|A|}A^*$.
- **方法二** 初等变换法. $\displaystyle(A \mid E) \xrightarrow{初等行变换} (E \mid A^{-1})$.
- **方法三** 用定义求 B, 使 $\displaystyle AB=E$ 或 $BA=E$, 则 A 可逆, 且 $A^{-1}=B$. 或令 $A=BC，则A^{-1}=C^{-1}B^{-1}$。往往需要用到各种变形、平方、乘积
- **方法四** 用[[2-8 分块矩阵|分块矩阵]].
