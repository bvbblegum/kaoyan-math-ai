## 向量和向量组的基本概念
- **定义**：$n$ 个数 $a_1, a_2, \cdots, a_n$ 组成的有序数组称为 $n$ 维向量。
- 线性组合 (线性表示)：
	- 定义：若 $\beta = k_1 \alpha_1 + k_2 \alpha_2 + \cdots + k_m \alpha_m$，则称 $\beta$ 可由向量组 $\alpha_1, \cdots, \alpha_m$ [[3-2 线性表示|线性表示]]。
	- 矩阵形式：$\beta = [\alpha_1, \cdots, \alpha_m] [k_1, \cdots, k_m]^T$，即 $Ax=\beta$ 有解。
	- **矩阵乘法的列向量理解**：矩阵 $A$ 右乘列向量 $x$，结果是 $A$ 的列向量的线性组合：$\begin{bmatrix}\alpha_{1},\alpha_{2},\alpha_{3} \end{bmatrix}\begin{bmatrix}k_1\\k_2\\k_3\end{bmatrix} = k_1\alpha_1 + k_2\alpha_2 + k_3\alpha_3$。
- 延伸组、缩短组：
	- $\alpha_1 = [a_{11}, a_{21}, \cdots, a_{r1}]^T, \alpha_2 = [a_{12}, a_{22}, \cdots, a_{r2}]^T, \cdots,\alpha_m = [a_{1 m}, a_{2 m}, \cdots, a_{rm}]^T$ 及 $\tilde{\alpha}_1 = [a_{11}, a_{21}, \cdots, a_{r1}, \cdots, a_{s1}]^T, \tilde{\alpha}_2 = [a_{12}, a_{22}, \cdots, a_{r2}, \cdots, a_{s2}]^T， \cdots, \tilde{\alpha}_m = [a_{1 m}, a_{2 m}, \cdots, a_{rm}, \cdots, a_{sm}]^T$
	- 其中 $s \ge r$，则称 $\tilde{\alpha}_1, \tilde{\alpha}_2, \cdots, \tilde{\alpha}_m$ 为向量组 $\alpha_1, \alpha_2, \cdots, \alpha_m$ 的**延伸组** (或称 $\alpha_1, \alpha_2, \cdots, \alpha_m$ 是 $\tilde{\alpha}_1, \tilde{\alpha}_2, \cdots, \tilde{\alpha}_m$ 的**缩短组**).
- 向量的内积：
	- 设 $\alpha = [a_1, a_2, \cdots, a_n]^T, \beta = [b_1, b_2, \cdots, b_n]^T$，则称 $(\alpha,\beta)=\alpha^{\intercal}\beta=\beta^{\intercal}\alpha=\sum_{i=1}^{n}a_{i}b_{i}\:.$ 为向量 $\alpha, \beta$ 的内积，记作 $(\alpha, \beta)$，即 $(\alpha, \beta) = \alpha^T \beta$。
- **向量的正交**：当 $\alpha^T \beta = 0$ 时，称向量 $\alpha, \beta$ 是正交向量。
	- $\alpha^T \alpha = 0$，则 $\alpha=0$
	- 三向量两两正交可推线性无关（两两线性无关不能推出三个线性无关）[[3-3 线性相关|线性相关]]
	- 三向量两两正交，则可做一个[[3-5 极大线性无关组|极大线性无关组]]，可做[[向量空间|向量空间]]的一组基
- 向量的模：
	- $\|\alpha\| = \sqrt{\sum_{i=1}^n a_i^2}$ 称为向量 $\alpha$ 的模 (长度)。
	- 当 $\|\alpha\| = 1$ 时，称 $\alpha$ 为**单位向量**。
- 标准正交向量组：
	- 若列向量组 $\alpha_1, \alpha_2, \cdots, \alpha_s$ 满足$\alpha_i^T \alpha_j = \begin{cases} 0, & i \ne j, \\ 1, & i=j, \end{cases}$则称 $\alpha_1, \alpha_2, \cdots, \alpha_s$ 为**标准**或**单位正交向量组**，也叫**规范正交基**。

### 等价向量组
- 定义：
	- 设两个向量组：<font color="#00b050">（I）</font> $\alpha_1,\alpha_2,...,\alpha_{\mathrm{n}},$<font color="#00b050">（II）</font>$\beta_1,\beta_2,...,\beta_{\mathrm{t}}$,若<font color="#00b050">（I）</font>中**每个向量** $a_{i}(i=1,2,...,s)$ 均可由 <font color="#00b050">（II）</font>中向量`线性表示`，则称向量组<font color="#00b050">（I）</font>可由向量组（II）[[3-2 线性表示|线性表示]]，从而 $\beta_1,\beta_2,...,\beta_{\mathrm{t}}$ 的`极大线性无关组`包含了 $\alpha_1,\alpha_2,...,\alpha_{\mathrm{n}},$ 的`极大线性无关组`
	- 若向量组（I）,（II）可以**相互线性表示**，则称向量组（I）与向量组（II）是`等价向量组`，记作（I） $\cong$（II）
- 向量组等价要同维，但向量个数可以不等。
	- 矩阵等价要同型，当然行数、列数都要相等；
- **必要**条件：
	- 如果向量组 (I) 和 (II) `等价`, 则 $r (I)=r (II)$.
- 充分必要条件：
> [!caution] 等价向量组的充分必要条件
> - $\alpha_{i}, \beta_{j} (i=1,2,\ldots,s; j=1,2,\ldots,t)$ **同维**，则 $\{\alpha_{1},\alpha_{2},\ldots,\alpha_{s}\} \cong \{\beta_{1},\beta_{2},\ldots,\beta_{t}\}$
>   - $\Leftrightarrow \{\alpha_{1},\alpha_{2},\ldots,\alpha_{s}\}$ 与 $\{\beta_{1},\beta_{2},\ldots,\beta_{t}\}$ 可以**相互表示**
>   - $\Leftrightarrow r(\alpha_{1},\alpha_{2},\ldots,\alpha_{s}) = r(\beta_{1},\beta_{2},\ldots,\beta_{t})$, **且一方可由另一方表示**,
>   - $\Leftrightarrow r(\alpha_{1},\alpha_{2},\ldots,\alpha_{s}) = r(\beta_{1},\beta_{2},\ldots,\beta_{t}) = r(\alpha_{1},\alpha_{2},\ldots,\alpha_{s},\beta_{1},\beta_{2},\ldots,\beta_{t})$ (==三秩相同==). [[3-6 向量组的秩|向量组的秩]]
> - 向量组和其`极大线性无关组`**等价**，向量组的`极大线性无关组`之间**等价**
> - 初等行变换后的行向量组与原行向量组等价。

## 线性相关
- ![[3-3 线性相关#线性相关与线性无关]]
	- **核心定理**：若线性无关的向量组 (I) 可由向量组 (II) 线性表示，则 (I) 的向量个数 $m$ $\le$ (II) 的向量个数 $n$。若一个线性无关的向量组可由另一个向量组线性表示，则前者的向量个数不多于后者的向量个数
- 性质
	1. 若向量组 $a_1,a_2,\cdots,a_n$ 线性无关，而 $\beta,a_1,a_2,\cdots,a_n$ 线性相关，则 $\beta$ 可由 $a_1,a_2,\cdots,a_n$ [[3-2 线性表示|线性表示]]。且表示法唯一 ^m 4 uh 1 i
	2. **部分**组 $\alpha_1,\alpha_2,\cdotp\cdotp\cdotp,\alpha_r$ **相关** $\Rightarrow$ **整体**组 $\alpha_1,\alpha_2,\cdotp\cdotp\cdotp,\alpha_r$, $\cdots,a$, 相关. 反之不成立
	3. **整体**组 $a_1,a_2,\cdots,a_r,\cdots,\alpha_{s}$,**无关** $\Rightarrow$ 任何**部分**组 $a_1,a_2,\cdots,a_r$ 无关。反之不成立
	4. $\alpha_1, \alpha_2, \cdots, \alpha_m$ 线性**无关** $\Rightarrow$ **延伸**组 $\tilde{\alpha}_1, \tilde{\alpha}_2, \cdots, \tilde{\alpha}_m$ 线性无关；反之不成立
	5. $\tilde{\alpha}_1, \tilde{\alpha}_2, \cdots, \tilde{\alpha}_m$ 线性**相关** $\Rightarrow$ **缩短**组 $\alpha_1, \alpha_2, \cdots, \alpha_m$ 线性相关，反之均不成立.
	6. 设有两个向量组（Ⅰ）$\alpha_1,\alpha_2,\cdots,\alpha_s$；（Ⅱ）$\beta_1,\beta_2,\cdots,\beta_t$：
		1. 若 $\beta_i$（$i=1,2,\cdots,t$）均可由（Ⅰ）线性表出，且 $t>s$，则（Ⅱ）$\beta_1,\beta_2,\cdots,\beta_t$ 线性相关。(**以少表多，多的相关**) [[3-2 线性表示#^1]]
		2. 若 $\beta_i$（$i=1,2,\cdots,t$）均可由（Ⅰ）线性表出，且 $\beta_1,\beta_2,\cdots,\beta_t$ 线性无关，则 $t\leqslant s$。
		3. 如：$\alpha_1 + \alpha_2, \quad \alpha_1 + 5\alpha_2, \quad 3\alpha_1 - 11\alpha_2$ 必相关
	- 两个向量相关->共线
	- 三个向量相关->共面
- 与线性表示关系
	- ![[3-3 线性相关#^m4uh1i]]
	- 若 $\beta$ 可由 $a_1,a_2,\cdots,a_n$ [[3-2 线性表示|线性表示]]，则表示法唯一 $\displaystyle\Leftrightarrow$$a_1,a_2,\cdots,a_n$ 线性无关

## 向量组的秩和极大线性无关组
### 极大线性无关组
- 定义：向量组里的**代表**
    - 设有向量组 $\alpha_1, \alpha_2, \cdots, \alpha_s$，如果在其中能选出 $r$ 个向量 $\alpha_{i_1}, \alpha_{i_2}, \cdots, \alpha_{i_r}$ 满足：
	    - 向量组 $\alpha_{i_1}, \alpha_{i_2}, \cdots, \alpha_{i_r}$ 线性无关；（**自身线性无关**）
	    - 向量组 $\alpha_1, \alpha_2, \cdots, \alpha_s$ 中每个向量均能被 $\alpha_{i_1}, \alpha_{i_2}, \cdots, \alpha_{i_r}$ 线性表示，(向量组中**任一向量皆可由其线性表示**)
	         - 也可表述成：向量组 $\alpha_1, \alpha_2, \cdots, \alpha_s$ 中任意 $r+1$ 个向量 (如果 $\alpha_1, \alpha_2, \cdots, \alpha_s$ 中有 $r+1$ 个向量的话) 都线性相关
	- 则称向量组 $\alpha_{i_1}, \alpha_{i_2}, \cdots, \alpha_{i_r}$ 是向量组 $\alpha_1, \alpha_2, \cdots, \alpha_s$ 的一个`极大线性无关组`
	- 极大线性无关组所含向量个数 $r$ 称为向量组 $\alpha_1, \alpha_2, \cdots, \alpha_s$ 的`秩`。
- 性质：极大无关组不唯一，但包含的向量个数唯一。
### 向量组的秩
- 定义：向量组 $\alpha_{{_{1}}},\alpha_{{_{2}}},\cdots,\alpha_{{_{s}}}$ 的[[3-5 极大线性无关组|极大线性无关组]] $\alpha_{{_{i}}},\alpha_{{_{i}}},\cdots,\alpha_{{_{i}}}$ 中所含向量的个数 $r$ 称为向量组的秩，记作$r(a_1,a_2,\cdots,a_s)=r.$
    - 如果 $\alpha_1, \alpha_2, \cdots, \alpha_s$ 全是零向量, 则规定 $r(\alpha_1, \alpha_2, \cdots, \alpha_s) = 0$. (此时没有最大无关组.)
    - 如果 (I) 是 $\alpha_1, \alpha_2, \cdots, \alpha_s$ 的[[3-5 极大线性无关组|极大线性无关组]], 则 (I) $\cong \alpha_1, \alpha_2, \cdots, \alpha_s$. [[等价向量组]]
    - 向量组的秩是刻画向量组**相关程度**的一个数量概念.
- 用秩判断[[3-2 线性表示|线性表示]]：
	- $$r(\alpha_1, \alpha_2, \cdots, \alpha_s, \beta) = \begin{cases} r(\alpha_1, \alpha_2, \cdots, \alpha_s), & \text{若 } \beta \text{ 可用 } \alpha_1, \alpha_2, \cdots, \alpha_s \text{ 表示,} \\ r(\alpha_1, \alpha_2, \cdots, \alpha_s) + 1, & \text{若 } \beta \text{ 不可用 } \alpha_1, \alpha_2, \cdots, \alpha_s \text{ 表示.} \end{cases}$$
	- $\beta$ 可用 $\alpha_1, \alpha_2, \cdots, \alpha_s$ `线性表示` $\Leftrightarrow r(\alpha_1, \alpha_2, \cdots, \alpha_s, \beta) = r(\alpha_1, \alpha_2, \cdots, \alpha_s)$.
	- $\beta$ 可用 $\alpha_1, \alpha_2, \cdots, \alpha_s$ `唯一线性表示` $\Leftrightarrow r(\alpha_1, \alpha_2, \cdots, \alpha_s, \beta) = r(\alpha_1, \alpha_2, \cdots, \alpha_s) = s$.
	- $\beta_1, \beta_2, \cdots, \beta_t$ 可以用 $\alpha_1, \alpha_2, \cdots, \alpha_s$ `线性表示` $\Leftrightarrow r(\alpha_1, \alpha_2, \cdots, \alpha_s, \beta_1, \beta_2, \cdots, \beta_t) = r(\alpha_1, \alpha_2, \cdots, \alpha_s)$.
	- $\displaystyle\alpha_1, \alpha_2, \cdots, \alpha_s \cong \beta_1, \beta_2, \cdots, \beta_t$ [[等价向量组]] $\Leftrightarrow r(\alpha_1, \alpha_2, \cdots, \alpha_s) = r(\alpha_1, \alpha_2, \cdots, \alpha_s, \beta_1, \beta_2, \cdots, \beta_t) = r(\beta_1, \beta_2, \cdots, \beta_t)$
	- `秩相等`是[[等价向量组]]的**必要条件**, 不是充要条件. 联想[[../5 特征值与特征向量/相似性质结论的辨析|相似矩阵]]
	- ![[3-2 线性表示#^2]]
- 用秩判断[[3-3 线性相关|线性相关]]
	- $r(\alpha_1, \alpha_2, \cdots, \alpha_s) = s \Leftrightarrow \alpha_1, \alpha_2, \cdots, \alpha_s$ 线性无关.

### 向量空间
- **定义**：$n$ 维向量的集合，对加法和数乘运算封闭。[[向量空间|向量空间]]
- **基与维数**：
	- **基**：空间中线性无关且能表示空间中任一向量的向量组（相当于极大无关组）。
	- **维数**：基中向量的个数，记为 $\dim V$。
- **坐标与基变换**：
	- **坐标**：向量 $\alpha$ 在基 $\epsilon_1, \dots, \epsilon_n$ 下的表示系数。
	- **过渡矩阵**：从基 $(\alpha_1, \dots, \alpha_n)$ 到基 $(\beta_1, \dots, \beta_n)$ 的变换：$(\beta_1, \dots, \beta_n) = (\alpha_1, \dots, \alpha_n)C$。
	- **坐标变换**：若 $\alpha$ 在旧基坐标为 $x$，在新基坐标为 $y$，则 $x = Cy$ 或 $y = C^{-1}x$。

## 内积与正交
- **内积**：$(\alpha, \beta) = \alpha^T \beta = \sum a_i b_i$。[[向量的内积与正交|向量的内积与正交]]
- **模 (长度)**：$||\alpha|| = \sqrt{(\alpha, \alpha)}$。
- **正交**：$(\alpha, \beta) = 0$。
- **施密特正交化** (Gram-Schmidt)：将线性无关组 $\alpha_1, \alpha_2, \alpha_3$ 转化为正交组 $\beta_1, \beta_2, \beta_3$：
	$$
    \begin{aligned}
    \beta_1 &= \alpha_1 \\
    \beta_2 &= \alpha_2 - \frac{(\alpha_2, \beta_1)}{(\beta_1, \beta_1)} \beta_1 \\
    \beta_3 &= \alpha_3 - \frac{(\alpha_3, \beta_1)}{(\beta_1, \beta_1)} \beta_1 - \frac{(\alpha_3, \beta_2)}{(\beta_2, \beta_2)} \beta_2
    \end{aligned}
    $$
    再将 $\beta_i$ 单位化得到标准正交基。[[施密特正交化|施密特正交化]]
- **正交矩阵**：$A^T A = E$ $\Leftrightarrow$ $A^{-1} = A^T$ $\Leftrightarrow$ 行 (列) 向量组为标准正交基。

## 线性方程组解的结构
- **基础解系**：齐次方程组 $Ax=0$ 的解空间的一组基。[[4-4 基础解系|基础解系]]
	- 向量个数 $s = n - r(A)$ （未知数个数 - 秩）。
	- 对应 $n-r(A)$ 个**自由变量**。
- **通解**：
	- 齐次通解：$k_1 \xi_1 + \dots + k_{n-r} \xi_{n-r}$。
	- 非齐次通解：$\eta^* + k_1 \xi_1 + \dots + k_{n-r} \xi_{n-r}$ （特解 + 齐次通解）。
