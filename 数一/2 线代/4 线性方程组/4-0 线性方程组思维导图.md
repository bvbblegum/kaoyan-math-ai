## 线性方程组的形式
- **矩阵式** $AX = \beta$ （齐次方程组 $AX = 0$）。
	- 于是 $n$ 维向量 $\xi$ 是 $AX = \beta$ 的解 $\Leftrightarrow A\xi = \beta$
	- $n$ 维向量 $\eta$ 是 $AX = 0$ 的解 $\Leftrightarrow A\eta = 0$

- **向量式**：$x_1\alpha_1 + x_2\alpha_2 + \cdots + x_n\alpha_n = \beta$
	- （齐次方程组 $x_1\alpha_1 + x_2\alpha_2 + \cdots + x_n\alpha_n = 0$）


## 解的情况判别
- **非齐次**线性方程组 $AX = \beta$
	- 有**唯一解** $\Leftrightarrow r(A \mid \beta) = r(A) = n$，==列满秩==，[[3-2 线性表示|线性表示]]
		- A 是方阵时有[[1-4 克拉默法则|克拉默法则]]
	- 无穷多解 $\Leftrightarrow r(A \mid \beta) = r(A) < n$，[[4-4 基础解系|基础解系]]
	- 无解 $\Leftrightarrow r(A \mid \beta) > r(A)$
	- 当A矩阵行满秩时，Ax=b必有解！
		- $m \ge r(A|\beta) \ge r(A) = m \implies r(A) = r(A|\beta) = m$
- **齐次**线性方程组 $AX = 0$
	- 只有**零解** $\Leftrightarrow A$ 列满秩 ($r(A)=n$)，[[3-3 线性相关|线性相关]]与线性无关
		- **推论**：A - $m \times n$ 阶，如 $r(A)=n$ （A **列满秩**）
			- 有左消去律 $AB=0 \implies B=0$，$AB=AC \implies B=C$
			- 则 $r(AB)=r(B)$，[[3-2 线性表示|线性表示]]
	- 有非零解 $\Leftrightarrow r(A) < n$，扁长条形方程组，[[4-4 基础解系|基础解系]]

## 通解的构造
### 齐次线性方程组 $AX = 0$
- 解集的性质：任何一组解的任意线性组合还是解
- [[4-4 基础解系|基础解系]]：解集的`极大无关组`（解空间的基）。
	- 如何**验证** $\eta_1, \eta_2, \cdots, \eta_t$ 是 $AX=0$ 基础解系
		1. 验证 $A\eta_i = 0$
		2. 证明 $\eta_1, \eta_2, \cdots, \eta_t$ **无关**
		3. 说明 $t = n - r(A)$
- `基础解系`中[[自由变量|自由变量]]个数：$n - r(A)$ （即自由变量的个数）。
	- [[基础解系、通解、自由变量与极大无关组的关系]]
- 通解公式：
	- 当 $\eta_1, \eta_2, \cdots, \eta_s$ 是 $AX = 0$ 的基础解系时，通解为：$X = c_1\eta_1 + c_2\eta_2 + \cdots + c_s\eta_s \quad (c_i \text{ 任意})$

### 非齐次线性方程组 $AX = \beta$
- 解集的性质：如果 $\eta_1, \eta_2, \cdots, \eta_s$ 是 $AX = \beta$ 的一组解
	- $c_1\eta_1 + c_2\eta_2 + \cdots + c_s\eta_s$ 也是 $AX = \beta$ 的一组解 $\Leftrightarrow c_1 + c_2 + \cdots + c_s=1$
	- $c_1\eta_1 + c_2\eta_2 + \cdots + c_s\eta_s$ 也是 $AX =0$ 的一组解 $\Leftrightarrow c_1 + c_2 + \cdots + c_s=0$
	- $AX = \beta$ 的两个解的差是 $AX = 0$ 的解。
	- 若 $\xi$ 是 $AX = \beta$ 的解，$\eta$ 是 $AX = 0$ 的解，则 $\xi + \eta$ 是 $AX = \beta$ 的解。
- 通解公式：$X = \xi_0 + c_1\eta_1 + c_2\eta_2 + \cdots + c_s\eta_s$
	- 其中 $\xi_0$ 是 $AX = \beta$ 的特解，$\eta_1, \cdots, \eta_s$ 是 $AX = 0$ 的基础解系，$c_i$ 可取任何常数。
- 解集的秩：$n - r(A)+1$


## 公共解与同解
### 公共解问题
- 联立方程/解
	- 若方程组（I）与方程组（II）已知，则联立方程组 $\begin{cases} (\text{I}) \\ (\text{II}) \end{cases}$ 的解就是方程组（I）与（II）的公共解。[[4-5 公共解问题|公共解问题]]
- 解代到方程里去
	- 求出方程组（I）的通解 $k_1\xi_1+k_2\xi_2+\cdots+k_s\xi_s$，代入方程组（II），求出 $k_i$ 之间的约束关系，代回方程组（I）的通解 $k_1\xi_1+k_2\xi_2+\cdots+k_s\xi_s$ 即为方程组（I）与（II）的公共解。
	- 若方程组（I）与方程组（II）没有给出，但若可求得方程组（I）的基础解系为 $\alpha_1, \alpha_2, \cdots, \alpha_s$，（II）的基础解系为 $\beta_1, \beta_2, \cdots, \beta_t$，则可设公共解为 $\gamma$，那么有 $\gamma=k_1\alpha_1+k_2\alpha_2+\cdots+k_s\alpha_s=l_1\beta_1+l_2\beta_2+\cdots+l_t\beta_t$，$k_1\alpha_1+k_2\alpha_2+\cdots+k_s\alpha_s-l_1\beta_1-l_2\beta_2-\cdots-l_t\beta_t=0$，解出 $k_1, k_2, \cdots, k_s$ 及 $l_1, l_2, \cdots, l_t$ 即可求得公共解 $\gamma$。
### 解的单向归属问题
- 设 $A, B$ 均为 $m \times n$ 矩阵，$Bx=0$ 的解都是 $Ax=0$ 的解
	- $\iff A$ 的行向量组可由 $B$ 的行向量组`线性表示`
	- $\iff r(B)=r\begin{pmatrix} A \\ B \end{pmatrix}$
	- $\iff Bx=0$ 与 $\begin{pmatrix} A \\ B \end{pmatrix}x=0$ 同解
- 区分：==A 被 B 线性表示，则 r（A）<r (B)，同时 B 的约束更多，$Bx=0$ 的解都是 $Ax=0$==
### 同解问题
- 聚焦于初等行变换，详见 [[4-3 同解方程组]]
- **齐次同解**：设 $A, B$ 均为 $m \times n$ 矩阵，$Ax=0$ 与 $Bx=0$ 同解
	- $\iff A$ 与 $B$ 的行`向量组等价`
	- $\iff r(A)=r(B)$，且 $Ax=0$ 的解满足 $Bx=0$
	- $\iff r(A)=r(B)=r\begin{pmatrix} A \\ B \end{pmatrix}$
	- $\iff Ax=0$ 的解满足 $Bx=0$，且 $Bx=0$ 的解满足 $Ax=0$

- **非齐次同解**：设 $A, B$ 均为 $m \times n$ 矩阵，$Ax=\alpha$ 和 $Bx=\beta$ 同解
	- $\iff (A,\alpha)$ 与 $(B,\beta)$ 行向量组等价
	- $\iff r(A,\alpha)=r(B,\beta)=r\begin{pmatrix} A, \alpha \\ B, \beta \end{pmatrix}$


> [!tip] 同解方程组的常用结论
> - 设矩阵 $P$ 列满秩，则 $PAx=0$ 与 $Ax=0$ 同解。
>     - PA 可由 A [[3-2 线性表示|线性表示]]，且秩相等
>     - 或者 AX 视为列向量，P 满秩乘 AX=0，只能 AX=0
> - $A^TAx=0$ 与 $Ax=0$ 同解。
>     - 若 $A^T A \alpha = 0$，则两边左乘 $x^T$：$x^T A^T A x = 0$，则 $(Ax)^T(Ax)=0$
>     - $r(A) = r(A^T) = r(A^T A) = r(AA^T)$
> - 设 $A$ 为 $n$ 阶矩阵，则 $A^nx=0$ 与 $A^{n+1}x=0$ 同解。

## 应用与联系
- **特征值与特征向量**
	- 属于 $n$ 阶矩阵 $A$ 的特征值 $\lambda$ 的特征向量就是齐次方程组 $(A - \lambda E)X = 0$ 的**非零解**。
	- 属于特征值 $\lambda$ 的线性无关特征向量组包含特征向量的个数 $= n - r(A - \lambda E)$。
