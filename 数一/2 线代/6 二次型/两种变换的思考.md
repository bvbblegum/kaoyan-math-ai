将二次型化为标准形是线性代数中的核心考点。图片中展示的两种方法：**[[数一\1 高数\0 汇总\方法与思维\正交变换|正交变换]]法**和**配方法**，虽然都能达到“化简二次型”的目的，但它们在数学原理、几何意义和计算结果上有着本质的区别。

以下是详细的对比解析：

### 1. 核心原理与变换矩阵的区别

| 比较维度         | [[数一\1 高数\0 汇总\方法与思维\正交变换|正交变换]]法                                                                   | 配方法                                                        |
|:---------------- |:---------------------------------------------------------------------------- |:------------------------------------------------------------- |
| **变换矩阵性质** | 使用**正交矩阵** $\boldsymbol{Q}$ ($\boldsymbol{Q}^T = \boldsymbol{Q}^{-1}$) | 使用可逆矩阵 $\boldsymbol{C}$ (仅要求 $\boldsymbol{C}\neq 0$) |
| **数学关系**     | 矩阵 $\boldsymbol{A}$ 与 $\boldsymbol{\Lambda}$ 既**合同**又**相似**         | 矩阵 $\boldsymbol{A}$ 与对角阵仅**合同**，通常不相似          |
| **坐标变换公式** | $\boldsymbol{x} = \boldsymbol{Qy}$                                           | $\boldsymbol{x} = \boldsymbol{Cy}$                            |

### 2. 标准形系数的含义（结果的区别）

这是两者最重要的数学区别：

*   **[[数一\1 高数\0 汇总\方法与思维\正交变换|正交变换]]法**：
    得到的标准形为 $f = \lambda_1 y_1^2 + \lambda_2 y_2^2 + \cdots + \lambda_n y_n^2$。
    *   **系数 $\lambda_i$ 必须是矩阵 $\boldsymbol{A}$ 的特征值**。
    *   结果是**唯一**的（不考虑特征值的排列顺序）。

*   **配方法**：
    得到的标准形为 $f = d_1 z_1^2 + d_2 z_2^2 + \cdots + d_n z_n^2$。
    *   **系数 $d_i$ 通常不是特征值**，它们只是运算过程中产生的一般常数。
    *   结果**不唯一**（取决于配方的顺序和提取的系数），**但符号（正负号）的个数是固定的**。

> **联系：** 根据“惯性定理”，无论用哪种方法，标准形中**正系数的个数（正惯性指数）和负系数的个数（负惯性指数）是相同的**。

### 3. 几何意义的区别

*   **[[数一\1 高数\0 汇总\方法与思维\正交变换|正交变换]]法（刚体运动）**：
    *   相当于对坐标轴进行了**旋转**或**镜面反射**。
    *   **保持长度和形状不变**。例如，如果原二次型方程 $f(x)=1$ 代表一个椭圆，变换后它依然是那个形状的椭圆，只是位置正了（主轴与坐标轴重合）。
    *   它保留了图形的度量性质（距离、角度）。

*   **配方法（仿射变换）**：
    *   相当于对坐标轴进行了**拉伸、压缩或剪切**。
    *   **改变了长度和形状**。例如，它可能会把一个扁长的椭圆拉伸成一个正圆。
    *   它不保留度量性质，只保留了拓扑性质（如曲面的类型）。

### 4. 计算难度与适用场景

*   **[[数一\1 高数\0 汇总\方法与思维\正交变换|正交变换]]法**：
    *   **步骤繁琐**：需要求特征值 $|\lambda \boldsymbol{I} - \boldsymbol{A}|$（解高次方程）、求特征向量、[[数一\2 线代\3 向量组\施密特正交化|施密特正交化]]、单位化。
    *   **适用场景**：题目明确要求“求正交变换化标准形”、涉及几何形状分析、或涉及特征值问题时。

*   **配方法**：
    *   **步骤简单**：仅涉及初等代数运算（完全平方公式），运算量通常较小。
    *   **适用场景**：题目仅要求“化为标准形”（未指定方法）、判断正定性、求惯性指数时。

### 5. 总结：图片底部思考题的答案

图片最下方的问题：**“思考：用[[数一\1 高数\0 汇总\方法与思维\正交变换|正交变换]]得到的标准形和用配方法得到的标准形一样吗？”**

**答案：通常不一样。**

*   **不一样的地方**：具体的系数数值不同（一个是特征值，一个是普通系数）。
*   **一样的地方**：项数相同，且非零系数的正负号个数相同（惯性指数相同）。

### 举例说明

设二次型 $f = x_1^2 + 4x_1x_2 + x_2^2$，对应矩阵 $\boldsymbol{A} = \begin{pmatrix} 1 & 2 \\ 2 & 1 \end{pmatrix}$。

1.  **用[[数一\1 高数\0 汇总\方法与思维\正交变换|正交变换]]法**：
    *   求特征值：$|\lambda \boldsymbol{I} - \boldsymbol{A}| = (\lambda-1)^2 - 4 = 0 \Rightarrow \lambda_1=3, \lambda_2=-1$。
    *   **标准形**：$f = 3y_1^2 - y_2^2$。

2.  **用配方法**：
    *   $f = (x_1 + 2x_2)^2 - 4x_2^2 + x_2^2 = (x_1 + 2x_2)^2 - 3x_2^2$。
    *   令 $z_1 = x_1+2x_2, z_2=x_2$。
    *   **标准形**：$f = z_1^2 - 3z_2^2$。

**对比**：
*   结果不同：一个是 $3y_1^2 - y_2^2$，一个是 $z_1^2 - 3z_2^2$。
*   惯性指数相同：都是一个正项，一个负项（正惯性指数 1，负惯性指数 1）。
