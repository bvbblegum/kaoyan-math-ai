## 题型汇总

### 事件的关系与运算
1. #随机事件与概率/事件的关系与运算 
   - 若 $A \subseteq B$，则 $A \cup B = B, A \cap B = A$;
   - $A \cup (B \cap C) = (A \cup B) \cap (A \cup C), A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$;
   - $\overline{A \cup B} = \overline{A} \cap \overline{B}, \overline{A \cap B} = \overline{A} \cup \overline{B}$;
   - 涉及复杂随机事件运算时，可考虑使用韦恩图，降低运算难度。

### 概率的性质与运用
2. #随机事件与概率/概率的性质与运用 
   - $P(A \cup B) = P(A) + P(B) - P(AB)$;
   - $P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)$;
   - $P(A-B) = P(A) - P(AB)=P(A \bar{B})$
	   - 若 $B \subseteq A$, 则 $P(AB) = P(B), P(A \cup B) = P(A), P(A - B) = P(A) - P(B), P(B - A) = 0$;
	   - $P(\bar{A}) = 1 - P(A)$;
   - 如 $P(A)=0$, 则 $P(ABC)=P(AB)=0$；如 $P(AB)=0$，则 $P(ABC)=0$
   - 涉及复杂随机事件的概率运算时, 可考虑使用韦恩图, 降低运算难度。
   - [[举例排除]]，**全、空、等、对立**

### 随机事件的独立性
5. #随机事件与概率/随机事件的独立性
	- $A$ 与 $B$ 相互独立 $$\begin{align*} &\Leftrightarrow P(AB) = P(A)P(B) \\&\Leftrightarrow P(A|B) = P(A|\bar{B}) = P(A) \quad (0 < P(B) < 1) \\ &\Leftrightarrow P(B|A) = P(B|\bar{A}) = P(B) \quad (0 < P(A) < 1) \\&\Leftrightarrow P(B|A) + P(\bar{B}|\bar{A}) = 1 \quad (0 < P(A) < 1);\end{align*}$$
	- **A, B, C 两两独立** $$\begin{cases} P(AB) = P(A)P(B), \\ P(AC) = P(A)P(C), \\ P(BC) = P(B)P(C). \end{cases}$$
	- **A, B, C 相互独立** $\Leftrightarrow$$$\begin{cases} P(AB) = P(A)P(B), \\ P(AC) = P(A)P(C), \\ P(BC) = P(B)P(C), \\ P(ABC) = P(A)P(B)P(C). \end{cases}$$
	- A 与 B 独立 $\Leftrightarrow$ A 与 $\bar{B}$ 独立 $\Leftrightarrow$ $\bar{A}$ 与 B 独立 $\Leftrightarrow$ $\bar{A}$ 与 $\bar{B}$ 独立
