## 随机试验、样本与随机事件

### 随机试验
随机试验具有如下特点：
1.  可以在相同的条件下重复进行；
2.  每次试验的可能结果不止一个，但能事先明确试验的所有可能结果；
3.  进行一次试验之前不能确定哪一个结果会出现。
则称此试验为**随机试验**，简称试验，通常用字母 $E$ 表示。

### 样本空间
- 随机试验 $E$ 的每个结果称为**样本点**，记作 $\omega$
- 所有可能结果组成的集合称为 $E$ 的**样本空间**，记为 $\Omega$

### 随机事件
- 样本空间的子集称为**随机事件**，简称事件，用字母 $A, B, C, \cdots$ 表示。
- 随机事件由样本点组成，其中由一个样本点组成的单点集，称为**基本事件**；由多个样本点组成的集合称为**复合事件**。
- 样本空间 $\Omega$ 包含所有的样本点，在每次试验中它总是发生的，称为**必然事件**。
- 空集 $\emptyset$ 不包含任何样本点，它也作为样本空间的子集，在每次试验中都不发生，称为**不可能事件**。
- 若一次试验的结果中，某一基本事件出现，则称该基本事件发生，对于某一事件 $A$，若其包含的某一样本点出现，则称事件 $A$ 发生。

> [!note] 注
> 事件是一个集合，因而事件间的关系与事件的运算自然按集合论中集合之间的关系和运算来处理。

## 事件间的关系与运算 

| 事件间关系与运算的语言描述                                    | 集合论中的表示法                                                                              | 概率论中的含义                                     |
| :----------------------------------------------- | :------------------------------------------------------------------------------------ | :------------------------------------------ |
| 事件 A 包含事件 B <br>(或事件 B 含于事件 A <br>或 B 发生导致 A 发生) | $A \supset B$ (或 $B \subset A$)                                                       | 事件 B 发生，则事件 A 一定发生                          |
| 事件 A 和 B 相等 <br>(或等价)                            | $A = B$                                                                               | 事件 A 发生，则 B 一定发生，反之亦然                       |
| 事件 A 与 B 之和 (或并)                                 | $A \cup B$ 或 $A + B$                                                                  | 事件 A, B 中至少有一个事件发生                          |
| 有限个事件 $A_1, A_2, \cdots, A_n (n \ge 2)$ 的和 (或并)  | $\displaystyle\bigcup_{i=1}^n A_i$ 或 $\displaystyle\sum_{i=1}^n A_i$                  | 事件 $A_1, A_2, \cdots, A_n$ 中至少有一个事件发生       |
| 可列个事件 $A_1, \cdots, A_n, \cdots$ 的和 (或并)         | $\displaystyle\bigcup_{i=1}^\infty A_i$ 或 $\displaystyle\sum_{i=1}^\infty A_i$        | 可列个事件 $A_1, \cdots, A_n, \cdots$ 中至少有一个事件发生 |
| 事件 A 与 B 的积 (或交)                                 | $A \cap B$ (简记为 $AB$)                                                                 | 事件 $A \cap B$ 发生，当且仅当 A 与 B 同时发生            |
| 有限个事件 $A_1, A_2, \cdots, A_n (n \ge 2)$ 的积 (或交)  | $A_1 \cap A_2 \cap \cdots \cap A_n$ <br> (简记为 $A_1 A_2 \cdots A_n$)                   | 事件 $A_1, A_2, \cdots, A_n$ 同时发生             |
| 可列个事件 $A_1, \cdots, A_n, \cdots$ 的积 (或交)         | $\displaystyle\bigcap_{i=1}^\infty A_i$                                               | 可列个事件 $A_1, \cdots, A_n, \cdots$ 同时发生       |
| 事件 A 与 B 的差                                      | $A - B=A-AB=A\bar{B}$                                                                 | 事件 $A - B$ 发生，当且仅当事件 A 发生而 B 不发生            |
| 事件 A 的逆事件 <br>(或**对立**事件)                        | $\bar{A}$ 或 $\Omega - A$<br>$A \cup B=\Omega$ 且 $A  B = \emptyset$                    | 事件 $\bar{A}$ 发生，当且仅当事件 A 不发生                |
| 事件 A 和 B 互不相容 (或 A 与 B **互斥**)                   | $A \cap B = \emptyset$                                                                | 事件 A 与 B 不可能同时发生                            |
| 事件 $A_1, \cdots, A_n$ 是一个**完备事件组**               | $A_1 \cup \cdots \cup A_n = \Omega$ <br> $A_i A_j = \emptyset, 1 \le i < j \le n$     | $n$ 个事件 $A_1, \cdots, A_n$ 中**仅发生且必发生其中之一** |
| 事件 $A_1, A_2, \cdots$ 是一个完备事件组                   | $A_1 \cup A_2 \cup \cdots = \Omega$ <br> $A_i A_j = \emptyset, 1 \le i < j < +\infty$ | 可列个事件 $A_1, A_2, \cdots$ 中仅发生且必发生其中之一       |



注
**三种关系**是指：包含，相等，互不相容；
**四种运算**是指：和，积，差，逆。(韦恩图)

-   **A, B 至少有一个发生** $\Leftrightarrow A \cup B$
-   **A, B 恰有一个发生** $\Leftrightarrow A\overline{B} \cup \overline{A}B$

$$
A \cup B = A\overline{B} \cup \overline{A}B \cup AB
$$

事件运算的一般顺序为:**“括号”一“逆”→“积”→“和”或“差**

> [!quote]
> A 和 B 至少有一个发生的前提下，A, B 恰有一个发生的概率为：$$P(A\overline{B} \cup \overline{A}B \ | \ A \cup B)$$

## 事件的运算性质
### 吸收律
若 $\displaystyle A \subset B$, 则 $\displaystyle A \cup B = B$, $\displaystyle AB = A$;
### 交换律
$\displaystyle A \cup B = B \cup A$, $AB = BA$;
### 结合律
$\displaystyle A \cup B \cup C = (A \cup B) \cup C = A \cup (B \cup C)$;
$\displaystyle ABC = (AB)C = A(BC)$;
### 分配律
$\displaystyle A(B \cup C) = AB \cup AC$;
$\displaystyle A \cup (BC) = (A \cup B)(A \cup C)$;
$\displaystyle (A \cup C)(B \cup C) = AB \cup C$;
$\displaystyle A \cap \left( \bigcup_{i=1}^{\infty} A_i \right) = \bigcup_{i=1}^{\infty} (A \cap A_i)$;
### 对偶律 (摩根律)
$\displaystyle \overline{A \cup B} = \bar{A} \bar{B}$;
$\displaystyle \overline{A \cap B} = \bar{A} \cup \bar{B}$;
$\displaystyle \overline{\bigcup_{i=1}^{n} A_i} = \bigcap_{i=1}^{n} \bar{A_i}$;
$\displaystyle \overline{\bigcap_{i=1}^{n} A_i} = \bigcup_{i=1}^{n} \bar{A_i}$;

> [!quote]
> *   $\overline{A \cup B}$: A 和 B 至少有一个发生的对立事件为“A, B 都不发生”，即 $\overline{A}\overline{B}$*   $\overline{AB}$: 
> * A 和 B 都发生的对立事件为“A, B 至少有一个不发生”，即 $\overline{A} \cup \overline{B}$
> * 尤其注意，$\overline{A}B \to A\overline{B} \to AB$，一定注意：$\overline{A}\overline{B} \ne \overline{AB}$
