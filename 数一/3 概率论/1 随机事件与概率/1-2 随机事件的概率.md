概率与长度、面 (体) 积、质量…是同一类概念，是一种度量。概率是事件 A 出现可能性大小的数值度量，记为 $P(A)$。
## 概率与条件概率的概念
### 概率的公理化定义
称 $P(A)$ 为事件 $A$ 的概率，如果 $P$ 满足下列条件：
1.  **非负性** 对于每一个事件 $A$，有 $P(A) \ge 0$;
2.  **规范性** 对于必然事件 $\Omega$，有 $P(\Omega) = 1$;
3.  **可列可加性** 设事件 $A_1, \cdots, A_n, \cdots$ 两两互不相容，即对于 $i \ne j, A_i A_j = \emptyset (i,j=1,2,\cdots)$，则 $\displaystyle P(A_1 \cup A_2 \cup \cdots) = P(A_1) + P(A_2) + \cdots, \text{即} P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i).$

## 概率的基本性质和基本公式
### 概率的基本性质
1. $P(\emptyset) = 0, P(\Omega) = 1, 0 \le P(A) \le 1.$.
2. 有限可加性设事件 $A_1, A_2, \cdots, A_n$ 两两互不相容，则$P(A_1 \cup A_2 \cup \cdots \cup A_n) = P(A_1) + P(A_2) + \cdots + P(A_n).$
3. **减法公式**：$P(A-B) = P(A) - P(AB)=P(A \bar{B})$;
   - 如果 $B \subset A$，则 $P(A-B) = P(A) - P(B), P(A) \ge P(B)$.
   - 如 $P(A)=0$, 则 $P(ABC)=P(AB)=0$
4. **求逆公式**：$P(\bar{A}) = 1 - P(A)$.
5. **加法公式**：
    - $P(A \cup B) = P(A) + P(B) - P(AB)$, 如 $AB$ 互斥，则 $P (AB)=0$
    - $P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(AB) - P(BC) - P(AC) + P(ABC)$.
    - 特别地，若 $A_1,A_2,\dots,A_n$ 两两互斥，即 $A_iA_j=\varnothing,\ i,j=1,2,\dots,n,\ i\ne j$，则 $\displaystyle P(A_1\cup A_2\cup\cdots\cup A_n)=P(A_1)+P(A_2)+\cdots+P(A_n).$

> [!note] 注
> 在计算一些事件的和的概率，即计算若干个事件中至少有一个事件发生的概率时，用概率的加法公式，但有时用对立事件的性质比较方便，特别是当这些事件相互独立的时候。

### 概率的基本公式
#### 乘法公式
$$
P(AB) = \begin{cases} P(A)P(B|A), & \text{当 } P(A)>0, \\ P(B)P(A|B), & \text{当 } P(B)>0. \end{cases}
$$
$$P(A_1 A_2 \cdots A_n) = P(A_1)P(A_2|A_1)P(A_3|A_1 A_2) \cdots P(A_n|A_1 \cdots A_{n-1})$$(当 $P(A_1 \cdots A_{n-1}) > 0$)

> 当 $A_i$ 先于 $A_{i+1}$ 发生时用乘法公式。

#### 全概率公式

如果事件 $A_1, \cdots, A_n$ 构成一个**完备事件组**，即它们两两互不相容，其和为 $\Omega$; 并且 $P(A_i)>0, i=1, 2, \cdots, n$, 则对任意事件 $B$, 有
$$
P(B) = \sum_{i=1}^n P(B A_i) = \sum_{i=1}^n P(A_i)P(B|A_i).
$$
特别地，上式中取 $n=2$, 并将 $A_1$ 记为 $A$, 此时 $A_2$ 就是 $\bar{A}$, 那么全概率公式成为
$$
P(B) = P(A)P(B|A) + P(\bar{A})P(B|\bar{A}).
$$

#### 贝叶斯公式
如果事件 $A_1, \cdots, A_n$ 构成一个**完备事件组**, 且 $P(A_i)>0, i=1, \cdots, n, P(B)>0$, 则对正整数 $m$, 有
$$
P(A_m | B) = \frac{P(B A_m)}{P(B)} = \frac{P(A_m)P(B|A_m)}{\sum_{i=1}^n P(A_i)P(B|A_i)} \quad (1 \le m \le n).
$$
特别地，上式中取 $n=2$, 并将 $A_1$ 记为 $A$, 此时 $A_2$ 就是 $\bar{A}$, 那么贝叶斯公式成为条件概率公式，即
$$
P(A|B) = \frac{P(AB)}{P(B)} = \frac{P(A)P(B|A)}{P(A)P(B|A) + P(\bar{A})P(B|\bar{A})}.
$$

> [!note] 注
> 全概率公式与贝叶斯公式使用的关键是要找到导致事件 B 发生的**完备事件组**。

## 概率的计算
### 1. 直接计算
在某些情况下，利用试验结果的等可能性，可以直接计算随机事件的概率。
#### 古典型概率
如果试验只有 $n$ 个基本事件，并且每一个基本事件出现的可能性相同，对于该试验的事件 A，则有
$$
P(A) = \frac{\text{A 所包含的基本事件个数 } k}{\text{基本事件的总数 } n} = \frac{k}{n}.
$$
- **特征**：$\Omega$ 中的样本点**有限**，且每个样本点发生的**可能性相同**。
- **公式**：$\displaystyle P(A) = \frac{m}{n}$，其中 $m$ 为 A 包含的基本事件数，$n$ 为 $\Omega$ 中的基本事件的总数。(*计数需注意：防重复、防漏计*)

> [!example] 常用结论 (超几何分布)
> 设 N 件产品中有 M 件次品，从中任取 n 件，则其中恰有 k 件次品的概率为：
> $$
> P(\text{恰有k件次品}) = \frac{C_M^k C_{N-M}^{n-k}}{C_N^n}
> $$
> 

#### 几何型概率
若试验的样本空间是几何区域 (直线、面积或体积)，其度量大小可用 $S_\Omega$ 表示，且任意点落在度量相同的子区域内是等可能的。当事件 A 为 $\Omega$ 的某一子区域，其度量为 $S_A$，则事件 A 的概率为
$$
P(A) = \frac{S_A}{S_\Omega}.
$$
- **特征**：$\Omega$ 中样本点**无限**且构成一个几何区域，且每个样本点的发生是**等可能**的。
- **公式**：$\displaystyle P(A) = \frac{L(A)}{L(\Omega)}$，其中 $L(A)$ 和 $L(\Omega)$ 分别表示 A 和 $\Omega$ 的几何测度（测度为长度或面积等）。

> [!example] 补例
> 向区间 $[0,1]$ 中随机投入一点：
>
> 1.  点**恰好**落在 $x=1/2$ 处的概率为 $0$。
>     -   结论：$P(A)=0 \not\Rightarrow A=\emptyset$ (概率为 0 的事件不一定是不可能事件)。
> 2.  点落在开区间 $(0,1)$ 内的概率为 $1$。
>     -   结论：$P(B)=1 \not\Rightarrow B=\Omega$ (概率为 1 的事件不一定是必然事件)。


> [!note] 注
> ① 古典概型与几何概型都有某种等可能性，或者说“均匀性”。两个概型的区别在于：几何概型的样本空间是无限 (不可列) 的，而古典概型的样本空间是有限的。
> ② 古典概型问题的主要计算工具是排列和组合，而几何概型问题的主要计算工具是用几何方法计算长度、面积等，也会用到**微积分计算**。
> ③ 古典概型的难点在于当选定样本空间后，正确计算事件 A 所**包含样本点的个数**；而几何概型难在**模型化** (如候车问题、会面问题等)：如何化为数学问题。

### 2.用频率估计概率
用事件 A 在大量重复试验中出现的频率 $\displaystyle f_n(A) = \frac{\text{A 在 n 次试验中出现的次数 } k}{\text{试验次数 } n}$ 作为概率 $P(A)$ 的近似值。

### 3.概率的推算
利用概率的基本性质、基本公式和事件的独立性，由简单事件的概率推算较复杂事件的概率。

### 4.利用概率分布计算概率
