## 事件的独立性
### 独立性的定义
1. **事件 A 与 B 独立** $\Leftrightarrow P(AB) = P(A)P(B)$
   - 事件 A, B 独立指 A 的发生与否对 B 发生的概率无影响。
   - 若 $P(B)>0 \implies P(A|B) = P(A)$
   - 若 $P(A)>0 \implies P(B|A) = P(B)$
   - 若 $P(A)=0 \implies P(AB)=0 = 0 \cdot P(B) = P(A)P(B)$

2.  **A, B, C 两两独立** $$
    \begin{cases} P(AB) = P(A)P(B), \\ P(AC) = P(A)P(C), \\ P(BC) = P(B)P(C). \end{cases}$$

3.  **A, B, C 相互独立** $\Leftrightarrow$$$
    \begin{cases} P(AB) = P(A)P(B), \\ P(AC) = P(A)P(C), \\ P(BC) = P(B)P(C), \\ P(ABC) = P(A)P(B)P(C). \end{cases}$$
==相互独立强于两两独立==

### 独立性的性质
1. A 与 B 独立 $\Leftrightarrow$ A 与 $\bar{B}$ 独立 $\Leftrightarrow$ $\bar{A}$ 与 B 独立 $\Leftrightarrow$ $\bar{A}$ 与 $\bar{B}$ 独立 (**四个独立性是等价的**)
2. $\Omega$ 与 $\emptyset$ 都与任意事件独立；
3. $A$ 与 $B$ 相互独立$$\begin{align*}
         &\Leftrightarrow P(AB) = P(A)P(B) \\
         &\Leftrightarrow P(A|B) = P(A|\bar{B}) = P(A) \quad (0 < P(B) < 1) \\
         &\Leftrightarrow P(B|A) = P(B|\bar{A}) = P(B) \quad (0 < P(A) < 1) \\
         &\Leftrightarrow P(B|A) + P(\bar{B}|\bar{A}) = 1 \quad (0 < P(A) < 1);
         \end{align*}$$
4. 若事件 $A_1, A_2, \dots, A_n$ 相互独立，则
    -  其中任意 $m (2 \le m \le n)$ 个事件也相互独立；
    -  其中任意一个事件与其佘任意 $m (2 \le m \le n)$ 个事件运算 (和、差与交) 也独立；
    -  将任意 $m (2 \le m \le n)$ 个事件分别换成其对立事件后所得 $n$ 个事件也独立.
5.  若 $P(B) \ne 0$, 则事件 A, B 相互独立的充要条件是 $P(A | B) = P(A)$;
6.  若 $P(B) \ne 0$, 则事件 A, B 相互独立的充要条件是 $P(A | B) = P(A | \bar{B})$;
7.  如果 $P(A) = 1$, 则对任意事件 B 都有 $P(AB) = P(A)P(B)$, 即 A 与 B 独立;
8.  如果 $P(A) = 0$, 则对任意事件 B 都有 $P(AB) = P(A)P(B)$, 即 A 与 B 独立;
9.  若 $A \subseteq B$, 则 A, B 独立的充要条件是 $P(A) = 0$ 或 $P(B) = 1$;
10. 若 A, B 互斥, 则 A, B 独立的条件是 $P(A) = 0$ 或 $P(B) = 0$.
### 独立与互斥的关系：
> [!tip] 关系辨析
> -   **互斥**：$A, B$ 有竞争关系，一个发生另一个就不能发生。
> -   **独立**：$A, B$ 没有关系，一个发生不影响另一个发生的概率。(*没有关系就没有影响*)

*   A 与 B 独立 $\Leftrightarrow P(AB)=P(A)P(B)$
*   A, B 互斥 $\Leftrightarrow AB = \emptyset \Leftrightarrow P(AB)=0$. (若 $P(AB) \ne 0$, 则 $AB \ne \emptyset$, A, B 不互斥)

但是当 $P(A)>0, P(B)>0$ 时，
* “A, B 独立” **推不出** “A, B 互斥”。
> 因为 $P(AB)=P(A)P(B)>0$, A, B 不互斥。
* “A, B 互斥” **推不出** “A, B 独立”。
> 因为 $AB = \emptyset \implies P(AB)=0 \ne P(A)P(B) > 0 \implies$ A, B 不独立。

## 独立重复事件与伯努利概型
- 将只有两个结果 $A$ 和 $\bar{A}$ 的随机试验称为**伯努利试验**；
- 若各次试验的结果相互独立称为**独立试验**；
- 将一伯努利试验独立重复的进行 $n$ 次称为 **$n$ 重伯努利试验**。

在每次试验中，事件 $A$ 发生的概率均为 $p$，则在 $n$ 次独立重复试验（或 $n$ 重伯努利试验）中，事件 $A$ 恰好发生 $k$ 次的概率为
$$
C_n^k p^k (1-p)^{n-k}, \quad k=0,1,2,\dots,n
$$

> [!note] 补充：负伯努利 (几何分布的推广)
> 在独立的重复伯努利试验中，第 $k$ 次试验恰好是 $A$ 第 $r$ 次发生的概率为 $C_{k-1}^{r-1} p^r (1-p)^{k-r}$。
