## 随机变量及其分布知识体系

### 随机变量与分布函数
- 分布函数的性质 (充要条件)
	- **单调性**：单调不减。
	- **有界性**：$0 \le F(x) \le 1$，且 $F(-\infty)=0, F(+\infty)=1$。
	- **右连续性**：$\lim_{x \to x_0^+} F(x) = F(x_0)$。
	- ==两分布相乘仍为分布==，两概率密度相乘则不一定
- 概率计算
	- $P\{a < X \le b\} = F(b) - F(a)$
	- $P\{X = a\} = F(a) - F(a^-)$ （对于连续型，此值为 0）

### 离散型随机变量
- **常见分布**
	- 0-1 分布 $B(1, p)$：$X$ 只取 0 或 1。
	- 二项分布 $B(n, p)$：$n$ 次伯努利试验中成功的次数。
		- $P\{X=k\} = C_n^k p^k (1-p)^{n-k}$
	- 泊松分布 $P(\lambda)$：稀有事件发生的次数。
		- $P\{X=k\} = \frac{\lambda^k}{k!}e^{-\lambda}$
		- 泊松定理：当 $n \to \infty, p \to 0, np = \lambda$ 时，==二项分布近似于泊松分布==。 [[泊松定理]]
	- 几何分布 $G(p)$：首中即止。
		- $P\{X=k\} = (1-p)^{k-1}p$ （具有**无记忆性**）。
	- 超几何分布：无放回抽样。
		- $P(X=k) = \frac{C_M^k C_{N-M}^{n-k}}{C_N^n}, \quad k=0,1,\cdots,\min\{M,n\}$
		- 超几何分布与二项分布的区别在于**超几何分布是无放回的抽样**, **二项分布是有放回的抽样**. 且当 $N \gg n$ 时, 超几何分布可**近似为**二项分布 $B\left(n, \frac{M}{N}\right)$。

### 连续型随机变量
- 概率密度 $f(x)$ [[2-2 离散型与连续型随机变量及其分布|连续型随机变量]]
	- 定义：$F(x) = \int_{-\infty}^x f(t)dt$。
	- ==性质==：$f(x) \ge 0$；$\int_{-\infty}^{+\infty} f(x)dx = 1$；$P\{x_1 < X \le x_2\} = \int_{x_1}^{x_2} f(x)dx$。
	- 与分布函数关系：在 $f(x)$ 连续点处，$F'(x) = f(x)$。
- 常见分布
	- 均匀分布 $U(a, b)$：
		- $f(x) = \frac{1}{b-a}, \quad a < x < b$。
	- 指数分布 $E(\lambda)$：
		- $f(x) = \lambda e^{-\lambda x}, \quad x > 0$。
		- 分布函数：$F(x) = 1 - e^{-\lambda x} \quad (x>0)$。
		- 性质：具有**无记忆性**。
	- **正态分布** $N(\mu, \sigma^2)$：
		- $f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$。
		- 标准正态分布 $N(0, 1)$：$\varphi(x) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}$。
		- 标准化：若 $X \sim N(\mu, \sigma^2)$，则 $Z = \frac{X-\mu}{\sigma} \sim N(0, 1)$。
		- 概率计算：$P\{a < X \le b\} = \Phi(\frac{b-\mu}{\sigma}) - \Phi(\frac{a-\mu}{\sigma})$。
		- **对称性**：$P\{\left|\frac{x-\mu}{\sigma}\right|\le Z\}=2\Phi(Z)-1$
- 常用结论：连续单增
- **离散型** $Y = g(X)$ [[2-3 随机变量函数的分布|随机变量函数的分布]]
	- 方法：合并相同的 $y_j$ 值对应的概率。
	  $$P\{Y=y_j\} = \sum_{g(x_i)=y_j} P\{X=x_i\}$$
- **连续型** $Y = g(X)$
	- **分布函数法 (定义法)**：适用于所有情况（万能方法）。
		1.  写出定义：$F_Y(y) = P\{Y \le y\} = P\{g(X) \le y\}$。
		2.  转化为 $X$ 的积分区域：$\int_{g(x) \le y} f_X(x)dx$。
		3.  求导得密度：$f_Y(y) = F_Y'(y)$。
	- **公式法**：仅适用于 $y=g(x)$ **严格单调**且可导的情况。
		- $f_Y(y) = f_X[h(y)] \cdot |h'(y)|$，其中 $h(y)$ 是 $g(x)$ 的反函数。
