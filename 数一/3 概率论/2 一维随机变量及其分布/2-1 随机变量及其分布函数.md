## 随机变量

### 随机变量定义
样本空间 $\Omega$ 上的单值实值函数 $X = X(\omega), \omega \in \Omega$, 其中 $\omega$ 为样本点。

> **理解**: 随机变量是一个映射，它将样本空间 $\Omega$ 中的每一个随机结果（样本点 $\omega$）映射到一个实数 $X(\omega)$。$\Omega \xrightarrow{X} \text{数}$
>

### 随机变量表示事件
设 $L$ 为任意实数集合, $\{X \in L\}$ 为随机事件 $\{-\infty \lt X \lt +\infty\} = \Omega$.

> **例如**：
> $\Omega = \{ \text{“出现反面”}, \text{“出现正面”} \} = \{H, T\}$
> 若定义随机变量 $X = \begin{cases} 0, & \omega = H, \ 1, & \omega = T, \end{cases}$

> [!note] 注：
> 随机变量是定义在**样本空间 $\Omega$** 上的，$\Omega$ 中的元素可以不是数；而普通函数是定义在**实数轴**上，变量的取值是数。
> 随机变量取值具有**随机性**且以一定的**概率**取值。

## 分布函数

### 分布函数的定义
对于随机变量 $X$, $x$ 是任意实数，则称函数
$$
F(x) = P(X \le x), \quad -\infty \lt x \lt +\infty
$$
为随机变量 $X$ 的分布函数。
> 几何意义：$X$ 落在区间 $(-\infty, x]$ 上的概率。

### 分布函数的性质
1.  **单调不减性**：$F(x)$ 是单调非减函数，即对任意的 $x_1 \lt x_2$，有 $F(x_1) \le F(x_2)$。
2.  **有界性**：对任意的 $x$，有 $0 \le F(x) \le 1$，且 $$F(-\infty) = \lim_{x \to -\infty} F(x) = 0, \quad F(+\infty) = \lim_{x \to +\infty} F(x) = 1.$$
3.  **右连续性**：$F(x)$ 是 $x$ 的右连续函数，即对任意的 $x_0$，有 $$\lim_{x \to x_0^+} F(x) = F(x_0), \quad \text{即} F(x_0+0) = F(x_0).$$

> [!tip] 
> $F(x)$ 成为分布函数的充要条件是性质 (1), (2), (3) 同时成立。

### 由分布函数表示的随机事件概率
设 $F(x)$ 为随机变量 $X$ 的分布函数，实数 $a\lt b$，则
1. $P\{x \le a\} = F(a)$, $P\{x \lt a\} = F(a^-)$
2. $P\{x=a\} = P\{x \le a\} - P\{x \lt a\} = F(a) - F(a^-)$
3. $P\{x \gt a\} = 1 - P\{x \le a\} = 1 - F(a)$.
4. $P\{a \lt x \le b\} = P\{x \le b\} - P\{x \le a\} = F(b) - F(a)$
5. $P\{a \le x \le b\} = P\{x \le b\} - P\{x \lt a\} = F(b) - F(a^-)$
6. $P\{a \lt x \lt b\} = P\{x \lt b\} - P\{x \le a\} = F(b^-) - F(a)$.
