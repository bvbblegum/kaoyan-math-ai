### 二维混合型随机变量
4. #二维随机变量及其分布/二维混合型随机变量 
	- 设 $X$ 的分布律为 $P\{X=x_i\} = p_i (i=1,2,\dots)$, Y 为连续型随机变量, $Z=g(X,Y)$, 则 Z 的分布函数可以用定义进行计算:$$F(z) = P\{Z \le z\} = P\{g(X,Y) \le z\} = \sum_i P\{X=x_i, g(x_i, Y) \le z\}$$其中 $P\{g(x_i, Y) \le z\}$ 可由 Y 的概率密度或者分布函数求得.此部分代入离散型随机变量的数值是必须的, 因为可以对二维随机变量实现降维求解. 但是运用`全概率公式`并不是必要的, 因为`条件概率`不一定是已知的.

### max 型和 min 型变量的分布函数
5. #二维随机变量及其分布/max型和min型变量的分布函数
	- 若 $(X,Y)$ 的联合分布函数为 $F(x,y)$. 定义 $U=\max\{X,Y\}, V=\min\{X,Y\}$ 这里需要区分两种情况:
	- ( i ) 若 $X,Y$ 相互独立, $U,V$ 的分布函数直接降维成一维分布函数,$$F_U(z) = P\{\max\{X,Y\} \le z\} = P\{X \le z, Y \le z\} = F_X(z)F_Y(z);$$$$\begin{aligned}F_V(z) &= P\{\min\{X,Y\} \le z\} = P\{(X \le z) \cup (Y \le z)\} \\&= 1 - P\{\min\{X,Y\} > z\} \\&= 1 - P\{X > z, Y > z\} \\&= 1 - [1-F_X(z)][1-F_Y(z)].\end{aligned}$$
	- ( ii ) 若 $X,Y$ 不相互独立, 仍然需要从随机事件的角度进行分析, 即 $$F_U(z) = P\{\max\{X,Y\} \le z\} = P\{X \le z, Y \le z\}$$$$ F_V(z) = P\{\min\{X,Y\} \le z\} = P\{(X \le z) \cup (Y \le z)\}$$
