## 二维均匀分布

### 二维均匀分布的定义

设 G 是平面上有界区域, 其面积为 A, 若二维连续型随机变量 $(X,Y)$ 的概率密度为
$$
f(x,y) = 
\begin{cases} 
\frac{1}{A}, & (x,y) \in G, \
0, & \text{其他},
\end{cases}
$$
则称 $(X,Y)$ 服从区域 G 上的均匀分布.

> [!tip] 二维均匀分布的性质
> 1. 概率与面积成正比. 即若 $(X,Y)$ 服从区域 $G$ (面积为 A) 上的均匀分布, $G_1$ 为区域 $G$ 的子区域, 即 $G_1 \subset G$, 且 $G_1$ 的面积为 $A_1$, 则 $$P\{(X,Y) \in G_1\} = \iint_{G_1} f(x,y) dx dy = \iint_{G_1} \frac{1}{A} dx dy = \frac{A_1}{A}.$$
> 2.  $(X,Y)$ 服从**矩形区域** $G = \{(x,y) | a\lt x\lt b, c\lt y\lt d\}$ 上的均匀分布的**充要条件**是 $X \sim U(a,b)$, $Y \sim U(c,d)$, 且 $X$ 与 $Y$ 相互独立.
> 3. 若 $(X,Y)$ 服从**圆形区域** $G=\{(x,y)|x^2+y^2 \le r^2\}$ 上的均匀分布, 则两个边缘分布都不是均匀分布, 且 X 与 Y 不独立, 但其**两个条件分布都是均匀分布**.

## 二维正态分布

若二维连续型随机变量 $(X,Y)$ 的概率密度为
$$
f(x,y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left\{-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_1)^2}{\sigma_1^2} - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right]\right\}
$$
其中 $\mu_1, \mu_2, \sigma_1, \sigma_2, \rho$ 为常数, 且 $\sigma_1\gt 0, \sigma_2\gt 0, |\rho|\lt 1$, 则称 $(X,Y)$ 服从参数为 $\mu_1, \mu_2, \sigma_1, \sigma_2, \rho$ 的正态分布, 记作 $(X,Y) \sim N(\mu_1, \mu_2; \sigma_1^2, \sigma_2^2, \rho)$.

> [!tip] 二维正态分布的性质
> 设 $(X,Y) \sim N(\mu_1, \mu_2; \sigma_1^2, \sigma_2^2, \rho)$, 则
> 1. 两个**边缘分布为正态分布**, 即 $X \sim N(\mu_1, \sigma_1^2)$, $Y \sim N(\mu_2, \sigma_2^2)$;
> 2. X 与 Y **相互独立的充分必要条件**是 $\rho=0$;
> 3. X 与 Y 的非零线性组合 $(aX+bY, cX+dY)$ 服从二维正态分布（$ad \neq bc$）;
> 4. X 与 Y 的线性组合 $aX+bY$ 仍为正态分布, 即 $$aX+bY \sim N(a\mu_1+b\mu_2, a^2\sigma_1^2 + b^2\sigma_2^2 + 2ab\rho\sigma_1\sigma_2)$$
> 	- 当 X 与 Y 相互独立时, 有 $$aX+bY \sim N(a\mu_1+b\mu_2, a^2\sigma_1^2 + b^2\sigma_2^2).$$
