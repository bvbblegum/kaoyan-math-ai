## 协方差和相关系数的思路

1. 先化简再计算，将复杂协方差拆成简单协方差

2. $$\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sqrt{D(X)}\sqrt{D(Y)}}$$

## 协方差

### 协方差的定义

设 $(X,Y)$ 为二维随机变量, 如果 $E\{[X-E(X)][Y-E(Y)]\}$ 存在, 则称其为随机变量 $X$ 和 $Y$的协方差, 记为 $\text{Cov}(X,Y)$, 即
$$
\text{Cov}(X,Y) = E\{[X-E(X)][Y-E(Y)]\}.
$$

### 协方差的计算公式

1. $\text{Cov}(X,Y) = E(XY) - E(X)E(Y)$;
2. $\text{Cov}(X,Y) = \rho_{XY} \sqrt{D(X)} \sqrt{D(Y)}$
3. $\text{Cov}(X_1+X_2, Y_1+Y_2) = \text{Cov}(X_1,Y_1) + \text{Cov}(X_1,Y_2) + \text{Cov}(X_2,Y_1) + \text{Cov}(X_2,Y_2)$
4. $\text{Cov}(aX, bY) = ab\text{Cov}(X,Y)$
5. $\text{Cov}(X,X) =E(X^2)-E(X)^2= D(X)$
6. 如果 $X$ 与 $Y$ 独立, 则 $\text{Cov}(X,Y)=0$.
7. $D(X \pm Y) = D(X) + D(Y) \pm 2\text{Cov}(X,Y)$.
8. $D(X_1 + X_2 + X_3) = D(X_1) + D(X_2) + D(X_3) + 2\text{Cov}(X_1, X_2) +  2\text{Cov}(X_1, X_3) + 2\text{Cov}(X_2, X_3)$

### 协方差的性质

1. $\text{Cov}(X,Y) = \text{Cov}(Y,X)$
2. $\text{Cov}(X,c) = 0, c$ 为任意常数;

## 相关系数

### 相关系数的定义

对于随机变量 $X$ 和 $Y$, 若 $D(X)\gt 0, D(Y)\gt 0$, 则称
$$
\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sqrt{D(X)}\sqrt{D(Y)}}
$$

### 相关系数的性质

1.  $|\rho_{XY}| \le 1$;
2.  $|\rho_{XY}|=1$ 的充分必要条件是: 存在常数 $a,b(a \ne 0)$ 使
$$
P\{Y=aX+b\}=1.
$$
且当 $a\gt 0$ (正相关) 时, $\rho_{XY}=1$; 当 $a\lt 0$ (负相关) 时, $\rho_{XY}=-1$.

### 随机变量不相关的定义

如果随机变量 $X$ 和 $Y$ 的相关系数 $\rho_{XY}=0 (\text{Cov}(X,Y)=0)$, 则称 $X$ 和 $Y$ **不相关**.

### 随机变量不相关的等价命题

对于随机变量 X 与 Y, 下面四个结论是等价的
1.  $\text{Cov}(X,Y) = 0$;
2.  $\rho_{XY}=0$, X 与 Y 不相关;
3.  $E(XY) = E(X)E(Y)$;
4.  $D (X \pm Y) = D (X)+D (Y)$.

> [!tip] 随机变量的独立性与不相关性
> 1. 若随机变量 $X$ 和 $Y$ **独立**, 则 $X$ 和 $Y$ **一定不相关**
> 2. 若 $X$ 和 $Y$ 不相关, 则不能推出 $X$ 和 $Y$ 相互独立;
> 3. 若 $X$ 和 $Y$ **相关**, 则一定推出 $X$ 和 $Y$ **不相互独立**
> 4. 若随机变量 $X$ 和 $Y$ 的联合分布是`二维正态分布`, 则 $X$ 和 $Y$ **独立等价于** $X$ 和 $Y$ **不相关**, 即 $\rho_{XY}=0$.
> 5. 若随机变量 X 与 Y 都服从 0-1 分布, 则 X 与 Y **独立的充要条件**是 X 与 Y **不相关**. ^in04mb
