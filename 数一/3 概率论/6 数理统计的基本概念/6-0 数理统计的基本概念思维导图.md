### 常用统计量及其数字特征
1. #数理统计的基本概念/常用统计量及其数字特征
	- 重要统计量的定义:$$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i, \quad S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2 = \frac{1}{n-1}\left(\sum_{i=1}^n X_i^2 - n\bar{X}^2\right).$$
	- `数字特征`: $E(\bar{X})=E(X), D(\bar{X})=\frac{D(X)}{n}, E(S^2)=D(X)$，
	- `正态分布`: 若 $X \sim N(\mu, \sigma^2)$, 则 $\displaystyle\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$.$\displaystyle E(S^2)=\sigma^2, D(S^2)=\frac{2\sigma^4}{n-1}, \text{Cov}(\bar{X}, S^2)=0$.[[统计量性质的证明]]

### 单正态总体的抽样分布
3. #数理统计的基本概念/单正态总体的抽样分布
	- $\displaystyle \bar{X} \sim N(\mu, \frac{\sigma^2}{n}), \frac{\bar{X}-\mu}{\sigma/\sqrt{n}} \sim N(0,1)$;
	- $\displaystyle \sum_{i=1}^n \left(\frac{X_i - \mu}{\sigma}\right)^2 \sim \chi^2(n)$;
	- $\displaystyle\sum_{i=1}^n \frac{(X_i - \bar{X})^2}{\sigma^2} = \frac{(n-1)S^2}{\sigma^2}= \frac{\left(\sum_{i=1}^n X_i^2 - n\bar{X}^2\right)}{\sigma^2} \sim \chi^2(n-1)$;[[关于样本方差的思考]]
		- $\bar{X}$ 与 $S^2$ 相互`独立`;
		- $\displaystyle E(S^2)=\sigma^2, D(S^2)=\frac{2\sigma^4}{n-1}, \text{Cov}(\bar{X}, S^2)=0$
	- $\displaystyle \displaystyle\frac{\frac{\bar{X}-\mu}{\sigma/\sqrt{n}}}{\sqrt{\frac{(n-1)S^2}{\sigma^2} / (n-1)}} = \frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1).$

### 双正态总体的抽样分布
4. #数理统计的基本概念/双正态总体的抽样分布
	- $\displaystyle \bar{X} \pm \bar{Y} \sim N\left(\mu_1 \pm \mu_2, \frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}\right), \quad \frac{(\bar{X}-\bar{Y})-(\mu_1-\mu_2)}{\sqrt{\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2}}} \sim N(0,1);$
	- $$\frac{(n_1-1)S_1^2}{\sigma_1^2} + \frac{(n_2-1)S_2^2}{\sigma_2^2} \sim \chi^2(n_1+n_2-2)$$$$ \frac{\frac{1}{\sigma_1^2}\sum_{i=1}^{n_1}(X_i-\mu_1)^2/n_1}{\frac{1}{\sigma_2^2}\sum_{i=1}^{n_2}(Y_i-\mu_2)^2/n_2} \sim F(n_1,n_2)$$$$\frac{\frac{(n_1-1)S_1^2}{\sigma_1^2}/(n_1-1)}{\frac{(n_2-1)S_2^2}{\sigma_2^2}/(n_2-1)} = \frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(n_1-1, n_2-1)$$
	- 若 $\sigma_1^2=\sigma_2^2=\sigma^2$, 则有$$T = \frac{\frac{(\bar{X}-\bar{Y})-(\mu_1-\mu_2)}{\sqrt{\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2}}}}{\sqrt{\left[\frac{(n_1-1)S_1^2}{\sigma_1^2}+\frac{(n_2-1)S_2^2}{\sigma_2^2}\right]/(n_1+n_2-2)}} = \frac{\bar{X}-\bar{Y}-(\mu_1-\mu_2)}{S_W\sqrt{\frac{1}{n_1}+\frac{1}{n_2}}} \sim t(n_1+n_2-2)$$其中 $S_W^2 = \frac{(n_1-1)S_1^2+(n_2-1)S_2^2}{n_1+n_2-2}$.
	- 此部分若直接记忆公式解题, 则难度大, 效率低。此部分解题主要是通过对复杂统计量中各个组成部分分别计算, 根据三大分布的定义逐步分析得到结果。
