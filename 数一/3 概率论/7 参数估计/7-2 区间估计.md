## 置信区间

设总体 $X$ 的分布函数 $F(x;\theta)$ 含有一个未知参数 $\theta$, 对于给定值 $\alpha (0\lt\alpha\lt1)$, 若由样本 $X_1, X_2, \dots, X_n$ 确定的两个统计量 $\theta_1=\theta_1(X_1, X_2, \dots, X_n)$ 和 $\theta_2=\theta_2(X_1, X_2, \dots, X_n)$ 满足
$$
P\{\theta_1 \lt \theta \lt \theta_2\} \ge 1-\alpha,
$$
则称随机区间 $(\theta_1, \theta_2)$ 是 $\theta$ 的置信水平为 $1-\alpha$ 的置信区间. $\theta_1$ 和 $\theta_2$ 分别称为置信水平 $1-\alpha$ 的双侧置信区间的置信下限和置信上限, $1-\alpha$ 称为置信水平.

## 单正态总体均值和方差的置信水平为 $1-\alpha$ 的区间估计

|    待估参数    |     其他参数      | 双侧置信区间                                                                                                                                     |
| :--------: | :-----------: | :----------------------------------------------------------------------------------------------------------------------------------------- |
|   $\mu$    | $\sigma^2$ 已知 | $\displaystyle\left( \bar{X} - u_{\alpha/2}\frac{\sigma}{\sqrt{n}}, \bar{X} + u_{\alpha/2}\frac{\sigma}{\sqrt{n}} \right)$                 |
|            | $\sigma^2$ 未知 | $\displaystyle \left( \bar{X} - t_{\alpha/2}(n-1)\frac{S}{\sqrt{n}}, \bar{X} + t_{\alpha/2}(n-1)\frac{S}{\sqrt{n}} \right)$                |
| $\sigma^2$ |   $\mu$ 已知    | $\displaystyle\left( \frac{\sum_{i=1}^n(X_i-\mu)^2}{\chi^2_{\alpha/2}(n)}, \frac{\sum_{i=1}^n(X_i-\mu)^2}{\chi^2_{1-\alpha/2}(n)} \right)$ |
|            |   $\mu$ 未知    | $\displaystyle\left( \frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)}, \frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)} \right)$                           |

## 两个正态总体均值和方差的置信水平为 $1-\alpha$ 的区间估计

|              待估参数               |                          其他参数                          | 双侧置信区间                                                                                                                                                                                                                                       |
| :-----------------------------: | :----------------------------------------------------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|          $\mu_1-\mu_2$          |              $\sigma_1^2, \sigma_2^2$ 已知               | $\left( \bar{X}-\bar{Y} - u_{\alpha/2}\sqrt{\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2}}, \bar{X}-\bar{Y} + u_{\alpha/2}\sqrt{\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2}} \right)$                                                      |
|                                 | $\sigma_1^2, \sigma_2^2$ 未知, 但 $\sigma_1^2=\sigma_2^2$ | $\left( \bar{X}-\bar{Y} - t_{\alpha/2}(n_1+n_2-2)S_w\sqrt{\frac{1}{n_1}+\frac{1}{n_2}}, \bar{X}-\bar{Y} + t_{\alpha/2}(n_1+n_2-2)S_w\sqrt{\frac{1}{n_1}+\frac{1}{n_2}} \right)$                                                              |
| $\frac{\sigma_1^2}{\sigma_2^2}$ |                   $\mu_1, \mu_2$ 已知                    | $\displaystyle\left( \frac{\frac{1}{n_1}\sum(X_i-\mu_1)^2}{\frac{1}{n_2}\sum(Y_i-\mu_2)^2} \frac{1}{F_{\alpha/2}(n_1,n_2)}, \frac{\frac{1}{n_1}\sum(X_i-\mu_1)^2}{\frac{1}{n_2}\sum(Y_i-\mu_2)^2} \frac{1}{F_{1-\alpha/2}(n_1,n_2)} \right)$ |
|                                 |                   $\mu_1, \mu_2$ 未知                    | $\displaystyle\left( \frac{S_1^2}{S_2^2} \frac{1}{F_{\alpha/2}(n_1-1,n_2-1)}, \frac{S_1^2}{S_2^2} \frac{1}{F_{1-\alpha/2}(n_1-1,n_2-1)} \right)$                                                                                             |

其中 $\displaystyle S_w^2 = \frac{(n_1-1)S_1^2+(n_2-1)S_2^2}{n_1+n_2-2}$.
